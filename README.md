# snu_solarcar_project
### Solar AI Mobility Challenge 서울대학교 공학기술과 사회 실습 프로젝트

**개요**
태양광 에너지 하베스팅과 AI 제어를 결합하여 최적의 주행 성능을 구현하는 실습.

**시스템 사양**
**MCU**: ESP32-C3 Mini (MicroPython)
**모터 드라이버**: DRV8833
**센서**: INA226 (전력 측정, I2C 0x40/0x41), MUX04 (라인 감지)
**액추에이터**: 패널 각도 조절용 서보모터

---
**Course**: 공학기술과 사회 (Engineering Technology and Society)
**Partner**: HandsOn Technology
